# perilla-judger-plugin-traditional
Judge plugin for perilla tradition problems(spj supported)

# Setup guide

Exec `sh install.sh`

When you see `/ #`, please manually install some packages:
 - `gcc`
 - `nodejs`
 - `python2`
 - `python3`
